# pdm_aulas_2023_1
Projeto com o que está sendo trabalhado em sala de aula.

# Instalação
Ao baixar o repositório, rode o comando pelo terminal para instalar todas as dependências:
```bash
$ yarn install
```

# Rodando o aplicativo
```bash
# Rodando no android
$ yarn android

# Rodando no ios
$ yarn ios

# Rodando empacotador metro
$ yarn start

# Rodando testes
$ yarn test

# Rodando linter
$ yarn lint
```
